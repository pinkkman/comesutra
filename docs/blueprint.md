# **App Name**: Fresher's Fiesta

## Core Features:

- Event Details Display: Clearly display event details such as date, time, and venue.
- RSVP/Registration: Allow attendees to easily RSVP and register for the event.
- Countdown Timer: Include a countdown timer to create anticipation for the party.
- FAQ Section: Provide a section to answer frequently asked questions about the event.
- Image Integration: Integrate the provided image into the website design.

## Style Guidelines:

- Primary color: Vibrant orange (#FFA500) to reflect the energy and excitement of the event.
- Background color: Light peach (#F9E79F), a very desaturated and brighter version of the orange hue.
- Accent color: Crimson red (#DC143C) for key interactive elements like the RSVP button and timer.
- Body and headline font: 'Poppins' sans-serif for a clean and contemporary look. Its geometric style can enhance readability while matching the vibrant aesthetic of the website.
- Use celebratory icons to represent different sections of the site.
- Use a dynamic and engaging layout that is consistent with the image used as a reference.
- Use subtle animations, such as animated confetti or smooth transitions, to enhance user engagement.
